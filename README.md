# ⛪ Site Institucional e Portal de Eventos

Este projeto consiste no desenvolvimento de uma plataforma web completa para uma instituição religiosa local, focada em presença digital e automação de processos.

## 🛠️ Tecnologias Utilizadas
* **Front-end:** HTML5, CSS3 e JavaScript.
* **Ícones:** Font Awesome.
* **Integração de Dados:** Google Sheets API (via automação de formulários).
* **Comunicação:** Integração direta com API do WhatsApp para envio de comprovantes.

## 🚀 Funcionalidades
* **Design Responsivo:** Interface otimizada para dispositivos móveis e desktops.
* **Gestão de Eventos:** Sistema de inscrição automatizado para o "Retiro Jovem".
* **Fluxo de Dados:** Captura de dados em tempo real enviando informações para planilhas de controle e validação de fluxo de caixa.
